## ğŸ“Œ What the project does

This Weather App enables users to:

- Search for particular locations and retrieve current weather information.
- View temperature, weather conditions, and multi-day forecasts
- Use autocomplete suggestions to quickly find locations
- Access the application directly through a browser with no setup required
- Add locations to a "Favourites" list, which can then be quickly accessed by the user.
